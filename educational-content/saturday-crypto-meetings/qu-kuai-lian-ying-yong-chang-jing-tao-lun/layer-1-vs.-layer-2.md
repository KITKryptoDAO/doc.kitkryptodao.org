# Layer 1 vs. Layer 2

### 0x00 定义

1. Layer 1 (Base Layer Protocols)
2. Layer 2 (Rollups)
3. Layer 3 (dApps)
4. Layer 0 (Cross-Chain)

### 0x01 使用场景

1. 主网扩容（吞吐量，数据可用性，交易速度...）
2. 支付通道
3. 未完待续...

2. Optimistic Rollups (e.g. [Optimism](https://community.optimism.io/), [Arbitrum](https://docs.arbitrum.io/))
